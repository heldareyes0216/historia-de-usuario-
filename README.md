# inventario python 

## este programa permite registrar un producto en un inventario. Le solicita al usuario el nombre del producto, el precio del producto y la cantidad para finalmente calcular el costo total.

---

## Funcionalidades:
1. solicitar el nombre del producto.
2. soliciatr el precio del producto y valida que el numero sea decimal.
3. solicitar la cantidad y valid que el numero sea entero.
4. se muestra el resumen de la informacion solicitada.
