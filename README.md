# Simple Inventory System

## Description

This project is a simple Python inventory program.  
It is a small application that registers a product, its price, and its quantity.

The program is an example of how basic Python concepts work in a practical scenario.

## How it works

1. The program asks the user to enter the name of a product.
2. The system requests the price of the product and validates the input.
3. The program asks for the quantity of the product.
4. The system checks if the entered values are valid numbers.
5. The program calculates the total cost by multiplying price and quantity.
6. Finally, the program displays a summary with the product information.
